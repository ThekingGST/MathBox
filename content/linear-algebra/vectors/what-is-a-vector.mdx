# What is a Vector?

In mathematics and physics, a **vector** is a quantity that has both *magnitude* and *direction*.

<Callout type="info" title="Formal Definition">
  A vector in $\mathbb{R}^n$ is an ordered list of $n$ real numbers.
</Callout>

## Notation

We write a vector $\vec{v}$ in column form:

$$
\vec{v} = \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}
$$

Or in row form: $\vec{v} = [1, 2, 3]$.

## In Code

```python
import numpy as np

# Create a vector
v = np.array([1, 2, 3])
print(v)          # [1 2 3]
print(v.shape)    # (3,)
```
